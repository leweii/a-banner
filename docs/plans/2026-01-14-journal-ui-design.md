# 手帐风格 UI 设计

## 概述

将现有 UI 改造为简约清新的手帐风格，包括手写字体、纸张质感、调色盘风格选择器等元素。

## 第 1 部分：整体风格与配色

### 整体氛围

简约清新手帐风，像翻开一本精心布置的笔记本。

### 配色方案

| 元素 | 颜色 |
|------|------|
| 页面背景 | 淡米色 `#FAF8F5` |
| 纸张/卡片 | 奶白色 `#FFFEF9` |
| 主文字 | 深灰棕 `#4A4543` |
| 网格线 | 淡蓝 `#E8EEF4` |
| 彩色高亮 | 粉 `#FFB5BA`、蓝 `#A8D8EA`、黄 `#FFEAA7`、绿 `#B8E6C1` |
| 按钮/强调 | 珊瑚色 `#FF8A80` |

### 纸张质感

- 主内容区：方格纸背景（淡蓝细网格）
- 卡片效果：微微倾斜的纸张阴影
- 角落装饰：折角效果或胶带贴纸感

## 第 2 部分：字体方案

### 中文手写字体

- 使用「站酷快乐体」(ZCOOL KuaiLe) - Google Fonts 免费提供
- 用于：页面标题、区块标题、按钮文字

### 英文手写字体

- 使用「Caveat」- Google Fonts 免费手写体
- 用于：英文内容、数字、副标题

### 代码/ASCII 展示

- 保持等宽字体，ASCII 预览需要保持字符对齐

### 字体加载方式

```tsx
// layout.tsx
import { ZCOOL_KuaiLe, Caveat } from 'next/font/google';

const zcool = ZCOOL_KuaiLe({ weight: '400', subsets: ['latin'] });
const caveat = Caveat({ subsets: ['latin'] });
```

## 第 4 部分：调色盘风格选择器

### 视觉设计

9 种风格排列成圆形调色盘，中间放置画笔图标。

### 风格颜色映射

| 风格 | 代表色 |
|------|--------|
| 水彩 | 淡蓝 `#A8D8EA` |
| 油画 | 深棕 `#8B7355` |
| 像素 | 亮绿 `#7FCD91` |
| 霓虹 | 紫红 `#FF6B9D` |
| 涂鸦 | 橙色 `#FFA07A` |
| 赛博朋克 | 青色 `#00D4FF` |
| 金属铬金 | 银色 `#C0C0C0` |
| 故障艺术 | 品红 `#FF00FF` |
| 火焰能量 | 橙红 `#FF4500` |

### 交互细节

- 悬浮时：色块放大 + 显示风格名称气泡
- 选中时：色块外圈发光 + 中间画笔图标指向该色块

### 实现方式

CSS 圆形布局 + transform 定位，hover 动画
